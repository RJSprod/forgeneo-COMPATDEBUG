# Forge Neo Compat Debugger

Temporary diagnostic-only extension for old Forge Neo builds. It logs compatibility option registration, option changes, config saves, UI component detection, runtime sampling state, and candidate internal patch targets while debugging `forgeNeo-compat`.

Logs are written to `logs/forge_neo_compat_debugger_YYYYMMDD_HHMMSS.txt` and mirrored to the terminal with the `[FNCDBG]` prefix.

Remove the extension by deleting this folder and restarting Forge Neo.
